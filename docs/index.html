How this monitor works

  • Fetches rural-relevant health policy news via RSS (e.g., KFF Health News, Rural Health Information Hub).
  • Uses a rule-based classifier to detect risk, opportunity, legislation/regulation, GME, and Deep South signals.
  • Builds a full daily digest (Markdown + HTML) and a homepage snapshot.
  • Runs entirely on GitHub Actions with no external API cost.
This tool is intended for educational and strategic planning purposes, not as official policy guidance.
